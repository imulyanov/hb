# 🎂 Бот для нагадування про дні народження

## Що вміє бот

- `/add Іван 15.06` — додати контакт
- `/list` — показати всіх
- `/delete Іван` — видалити контакт
- Кожного дня о **09:00** перевіряє чи є дні народження через 7 днів і надсилає тобі нагадування

---

## Крок 1 — Створи Telegram-бота

1. Відкрий Telegram і знайди **@BotFather**
2. Надішли команду `/newbot`
3. Введи назву бота (наприклад: `Мої дні народження`)
4. Введи username бота (має закінчуватись на `bot`, наприклад: `moi_dni_bot`)
5. BotFather надішле тобі **токен** — рядок типу `7123456789:AAH...`
   → **Збережи його, він знадобиться!**

---

## Крок 2 — Дізнайся свій Chat ID

1. Знайди свого нового бота в Telegram і натисни `/start`
2. Надішли команду `/myid`
3. Бот відповість твоїм **Chat ID** — число типу `123456789`
   → **Збережи його теж!**

> ⚠️ Ця команда запрацює тільки після повного розгортання на Railway (Крок 4).
> Альтернатива: відкрий посилання в браузері (підстав свій токен):
> `https://api.telegram.org/bot<ТВІЙ_ТОКЕН>/getUpdates`
> Напиши боту `/start`, потім перезавантаж сторінку — побачиш `"chat":{"id":ЧИСЛО}`

---

## Крок 3 — Завантаж файли на GitHub

1. Зайди на [github.com](https://github.com) і створи акаунт (якщо немає)
2. Натисни **New repository**, назви його `birthday-bot`, зроби **Private**, натисни **Create**
3. Завантаж усі 4 файли (`bot.py`, `requirements.txt`, `Procfile`, `railway.toml`):
   - Натисни **Add file → Upload files**
   - Перетягни всі 4 файли
   - Натисни **Commit changes**

---

## Крок 4 — Розгорни на Railway (безкоштовно)

1. Зайди на [railway.app](https://railway.app) → **Login with GitHub**
2. Натисни **New Project → Deploy from GitHub repo**
3. Вибери свій репозиторій `birthday-bot`
4. Railway автоматично розпочне деплой

### Додай змінні середовища:

5. У проєкті натисни на сервіс → вкладка **Variables**
6. Додай два значення:

   | Name | Value |
   |------|-------|
   | `BOT_TOKEN` | `7123456789:AAH...` (твій токен від BotFather) |
   | `CHAT_ID` | `123456789` (твій Chat ID) |

7. Натисни **Deploy** або Railway зробить це автоматично

### Додай постійне сховище (щоб контакти не зникали):

8. У проєкті натисни **+ Add** → **Volume**
9. Mount Path: `/app` (або залишити за замовчуванням)
10. Натисни **Create Volume**

---

## Крок 5 — Перевір роботу

1. Знайди свого бота в Telegram
2. Надішли `/start` — бот має відповісти
3. Спробуй `/add Тест 01.01`
4. Спробуй `/list` — побачиш доданий контакт
5. Спробуй `/delete Тест`

Якщо все працює — бот активний 24/7 і не залежить від твого комп'ютера! 🎉

---

## Як використовувати

```
/add Мама 12.03        → додає Маму з д.н. 12 березня
/add Олег 05.11        → додає Олега з д.н. 5 листопада
/list                  → показує всіх
/delete Олег           → видаляє Олега
```

Бот щодня о 09:00 (за UTC, тобто приблизно 11:00-12:00 за Київським часом)
перевіряє список і надсилає нагадування за 7 днів до дня народження.
